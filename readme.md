# 🛡️ AI Message Safety Analyzer

A hybrid **Machine Learning + LLM based spam detection system** that analyzes messages and identifies potential spam or phishing content.

Built using **Python, NLP, Streamlit, and Groq LLM**.

---

## 🚀 Features

* 🔍 Single Message Spam Detection
* 📂 Bulk File Scanner
* 🧠 AI Explanation using LLM
* 📊 Spam Probability Score
* ☁️ WordCloud Visualization
* ⚠️ Suspicious Word Detection
* 📈 Spam Trend Analytics
* 📁 Supports Multiple File Formats

Supported formats:

* CSV
* TXT
* Excel (.xlsx)
* JSON
* PDF

---

## 🧠 Tech Stack

* Python
* Scikit-learn
* NLP (CountVectorizer)
